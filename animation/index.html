Animation improvements

Below are my animation improvements. The recordings might not be able to capture them fully due to the frame rate and poor luck.

1. Added red flash to the like button

Now the like button flashes when you like a message for additional feedback.

2. Scale profile picture on hover

Now the profile pictures scale up when you hover over them for better visibility but also not taking up too much space initially.

3. Scale edit input on appearance

Now the edit message controls render with a scaling animation when they are triggered for better feedback.

4. Border flash on newly edited message

Now when a message is edited for the first time there's an animation which creates a temporary border around the message.

5. Enabled buttons scale up and down

Now when a button becomes enabled it scales up and down for increased feedback such as when the user satisfies the requirement for changing the picture.
